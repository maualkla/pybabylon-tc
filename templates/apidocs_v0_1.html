Login Service GET /login

Service called via a GET request. Serve as a login service.

Request params:

  • u : Username string
  • p : Base64 password string

Response payload:

{ "expire": expire_string, "id": id_string, "username": username_string }

Error responses:

  • 401 : User or password not valid.
  • 404 : User not yet registered.

Signup Service POST /signup

Service called via a POST request.

Request payload:

{ "activate": active_bool, "username": username_string, "bday": birthday_string, "email": email_string, "fname": fullname_string, "pass": b64password_string, "phone": phone_string, "pin": pin_string, "plan": plan_number, "postalCode": postalcode_string, "terms": terms_bool, "type": type_nuber }

Response payload:

{ "trxId": transaxtion_id }

Error responses:

  • 500 : Error creating user in backend.
  • 409 : User already registered.
  • 400 : Bad request.
